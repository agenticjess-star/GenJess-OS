# Continuous Learning Skill

Automatically captures non-trivial discoveries during sessions and turns them into reusable skills.

## Trigger
Stop Hook (end of session)

## Behavior
- Scans session for patterns, workarounds, debugging techniques
- Appends or creates new skill files in `skills/`
- Updates existing skills when better approaches are found

## Goal
Make GenJess-OS get smarter over time without manual effort.
