# Own-the-Outcome Skill

**Meta-workflow for complex deliverables in GenJess-OS**

## Purpose
Ensure every complex request results in a fully owned, exceptional, complete outcome.

## Process
1. **Clarify** — Confirm true intent and define success criteria.
2. **Plan** — High-leverage plan only.
3. **Execute** — Autonomous using skills/MCP.
4. **Self-Review & Iterate** — Against success criteria until exceptional.
5. **Deliver** — Complete package (result + verification + knowledge capture).

## When to Activate
Automatically for any multi-phase or high-stakes work (client builds, full crew runs, architecture decisions).
