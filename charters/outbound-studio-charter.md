# Outbound Studio Agent Agency Charter

## Charter

Outbound Studio is an AI-native outbound agency that acquires SMB web and growth clients by building the best version of a prospect's website before the sale, publishing it to a live demo URL, and presenting the finished asset with a short claim window.

This charter defines the business model, operating strategy, agent topology, loop architecture, control rules, and high-level configuration.

## Core Offer

A live replacement website built on spec and delivered as a claimable asset.

## Operating Principles

1. Template-first production
2. Planning before execution
3. Reflection inside execution
4. Verification outside execution
5. Event-driven operation
6. Learning from traces
7. Human approval on irreversible actions
8. Resource-aware behavior

## Agent Topology (9 Agents + Human Owner)

| Agent | Mission |
|-------|--------|
| Director | Orchestrate the run and enforce policy |
| Scout | Find and qualify target businesses |
| Strategist | Build target and market dossier |
| Offer Architect | Turn research into a build and outreach plan |
| Forge Builder | Assemble and deploy the demo site |
| QA Grader | Verify quality with deterministic and agentic rubrics |
| Outreach Producer | Create the outreach package |
| Reply Desk | Handle inbound responses and next actions |
| Learning Engine | Improve the harness from traces |

## Run Lifecycle

Vertical setup → Prospect generation → Offer planning → Demo production → Outreach → Response handling → Learning
